# Gopal Krishna — Portfolio

Personal portfolio website built with React + Vite + Tailwind CSS.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Framework: Vite (auto-detected)
4. Click Deploy — done!

Your live link will be: `https://your-repo-name.vercel.app`
